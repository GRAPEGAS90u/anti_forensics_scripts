# anti_forensics_scripts

Welcome to my digital lab! I’m a beginner on my journey into the worlds of Cyber Security and IoT. I’m taking this seriously—well, at least enough to start building my own tools and hardening my system.

### About this Repository

- This is not a collection of professional-grade security software. Instead, it’s a living notebook of the scripts I’ve written to harden my Linux Mint setup and experiment with anti-forensics.

# ! Keep in mind !

Not Masterpieces: These are beginner-level scripts. They are functional and meant for learning, but they are definitely works in progress.

My Workflow: I use Linux Mint and Zsh. Most of these scripts are tailored to my specific setup and might need adjustment for other environments.

Serious, but Experimental: I’m serious about learning how systems work under the hood, but I’m still having fun with the „grind.“

What you’ll find here
Hardening Scripts: Tweaks to restrict Flatpak permissions, D-Bus communication, and general system hardening.

Obfuscation & Anti-Forensics: Tools for zeroing out free space, secure deletion, and cleaning up system caches.


# A Warning

If you decide to use these scripts: Be careful! Commands that involve secure deletion or clearing caches can have side effects. Always make sure you understand exactly what a command does—especially when you use sudo—before running it on your own machine.

Feedback
Since I’m still learning, I’m open to suggestions. If you find a bug, see a security flaw, or have a better way to do things, feel free to open an issue or send a PR. I’m here to learn and improve!
